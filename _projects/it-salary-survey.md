### Project Overview
An in-depth explorative data analysis (EDA) of the IT Salary Survey in the EU. This project uncovers trends in compensation, distribution of roles, and gender pay gaps within the tech industry in Europe.

### Key Features
- **Data Visualizations**: High-quality charts using Matplotlib and Seaborn.
- **Statistical Analysis**: Insights into salary distributions.
- **Trend Identification**: Identifying key factors affecting IT salaries.
