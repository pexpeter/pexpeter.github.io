### Project Overview
This project uses Deep Learning to classify images of animals. A Convolutional Neural Network (CNN) was trained to recognize 80 different animal species with high accuracy.

### Key Features
- **Deep Learning**: Customized CNN architecture for image recognition.
- **Flask API**: REST API to serve predictions.
- **Deployment**: Hosted on Vercel for easy access.
